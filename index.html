NMRAnet Web Site

(Content not yet final)

The purpose of the NMRAnet project is to develop a new Layout Control Bus for use on Model Railroads that could be adopted as a Standard by the NMRA.

There are two primary NMRAnet development projects, S9.5 and S9.6. More information is available below.

The most recent work is available via the NMRAnet development Wiki main page. See in particular the work on acceptance criteria.
